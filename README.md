# CitySense

**CitySense** is a modern mobile application designed to empower citizens to actively participate in keeping their cities clean. By simplifying the process of reporting waste and providing a transparent platform for tracking issues, CitySense bridges the gap between citizens and municipal services.

> **"Empowering citizens to build cleaner, smarter cities."**
